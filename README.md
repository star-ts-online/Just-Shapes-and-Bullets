# Just-Shapes-and-Bullets
A short 2D shooter made in Godot. My first complete gamedev project.

Rules are simple: 
- You're a small square trapped in a box for 3 minutes.
- You can move with WASD and shoot with left mouse.
- There's gonna be triangle-shaped enemies trying to get you. Shoot 'em to survive 'til the time limit.
